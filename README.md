# Gestão de clientes e portfólio

Sistema em desenvolvimento para gestão de clientes e acompanhamento de portfólio, com foco em organização de dados, visualização de informações financeiras e evolução para um sistema completo de investimentos.

---

## Objetivo do projeto

Este projeto tem como objetivo simular um sistema de gestão de clientes e seus respectivos dados de portfólio, permitindo:

- Visualização de clientes
- Organização de dados cadastrais
- Associação de clientes com gestores e distribuidores
- Evolução futura para análise de portfólio e investimentos

---

## Motivação

Este projeto está sendo desenvolvido como parte do processo de aprendizado e evolução técnica em desenvolvimento de software, com foco em sistemas financeiros.

A ideia é simular um ambiente próximo ao utilizado em instituições financeiras, reforçando boas práticas de arquitetura, organização de dados e integração entre sistemas.

---

## Tecnologias utilizadas

- React.js (Frontend)
- Node.js (Backend para controle de clientes)
- JavaScript (ES6+)
- HTML5
- CSS3
- Postman (testes de API)
- Git & GitHub

---
